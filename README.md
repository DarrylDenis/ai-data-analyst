# AI Data Analyst (Client-Side Version)

This is a browser-based Data Analytics application. It runs entirely in the browser using React, PapaParse (for CSV processing), and Recharts.

## Features
1. **Data Upload**: CSV/Excel parsing in the browser.
2. **Profiling**: Automated column type detection and quality scoring.
3. **Cleaning**: AI-suggested cleaning plans executed via JavaScript.
4. **Statistics**: Client-side calculation of Mean, Median, StdDev, and Correlations.
5. **Visualization**: Dynamic charts generated by Recharts.

## How to Use
1. No installation required.
2. The app uses `importmap` in `index.html` to load dependencies from CDNs.
3. Add your Gemini API Key in the code or environment settings if using AI features.
